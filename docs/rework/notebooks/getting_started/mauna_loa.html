Case Study: \(CO_2\) Levels#

As a case study we will look at :math:`CO_2 levels, as recorded by the Mauna Loa laboratory <https://gml.noaa.gov/ccgg/trends/>`__.

First we’ll download and extract the data:

[1]:

from copy import deepcopy

import matplotlib.pyplot as plt
import numpy as np
import pandas as pd
from matplotlib.axes import Axes

import gpflow

[2]:

co2_data = pd.read_csv(
    "https://gml.noaa.gov/webdata/ccgg/trends/co2/co2_mm_mlo.csv", comment="#"
)
co2_data = co2_data[co2_data["decimal date"] > 1990]
Xco2 = co2_data["decimal date"].values[:, None]
Yco2 = co2_data["average"].values[:, None]

And then we’ll define a helper function to plot our kernel against the \(CO_2\) data:

[3]:

def plot_co2_model_prediction(
    ax: Axes, model: gpflow.models.GPModel, start: float, stop: float
) -> None:
    Xplot = np.linspace(start, stop, 200)[:, None]
    idx_plot = (start < Xco2) & (Xco2 < stop)

    f_mean, f_var = model.predict_f(Xplot, full_cov=False)
    f_lower = f_mean - 1.96 * np.sqrt(f_var)
    f_upper = f_mean + 1.96 * np.sqrt(f_var)

    ax.plot(Xco2[idx_plot], Yco2[idx_plot], "kx", mew=2)
    (mean_line,) = ax.plot(Xplot, f_mean, "-")
    color = mean_line.get_color()
    ax.plot(Xplot, f_lower, "--", color=color)
    ax.plot(Xplot, f_upper, "--", color=color)
    ax.fill_between(Xplot[:, 0], f_lower[:, 0], f_upper[:, 0], color=color, alpha=0.1)


def plot_co2_kernel(kernel: gpflow.kernels.Kernel) -> None:
    model = gpflow.models.GPR(
        (Xco2, Yco2),
        kernel=deepcopy(kernel),
    )
    opt = gpflow.optimizers.Scipy()
    l = opt.minimize(
        model.training_loss, model.trainable_variables, options=dict(maxiter=1000)
    )
    # print(l)
    # gpflow.utilities.print_summary(model, "notebook")

    fig, (ax1, ax2) = plt.subplots(nrows=1, ncols=2)
    plot_co2_model_prediction(ax1, model, 1950, 2050)
    plot_co2_model_prediction(ax2, model, 2015, 2030)

Let’s try with our faouvorite SquaredExponential kernel first:

[4]:

plot_co2_kernel(gpflow.kernels.SquaredExponential())

On the left side is a plot of our entire time range. On the right side we zoom in to the 2015-2030 range.

Our model clearly picks up the general trend of the data, but I see two problems: * It doesn’t pick up the periodic anual trend. * When we look at our predictions for 2050, it doesn’t seem to continue the upwards trend of the data.

[5]:

plot_co2_kernel(
    gpflow.kernels.SquaredExponential(variance=25000, lengthscales=130)
    + gpflow.kernels.Periodic(gpflow.kernels.SquaredExponential())
